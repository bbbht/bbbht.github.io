发表于 2018-03-20 | 分类于 zsh | | 总阅读量

已经习惯使用zsh来代替bash
zsh提供了大量有用且使用方便的插件和主题
默认情况下就开箱即用，配合oh-my-zsh更有了极大的可折腾性
目前主要使用插件来提供以下功能：

  • 自动补全（常规补全及历史提示）
  • 语法高亮（错误提示）
  • 状态提醒（git以及命令执行错误码）
  • 路径跳转

git 插件会影响终端响应速度，尤其当项目较大

另外像take,d等命令也很给力
